# CyberFocus 个人中控台

一个集时间管理、任务规划、专注计时于一体的现代化个人效率中心平台。

## 功能特色

### ⚡ 快速访问
- 自定义常用网站快捷方式
- 支持编辑模式，可添加、修改或删除快捷链接
- 玻璃拟态设计风格，美观实用

### 📝 三层任务管理系统
- **待办事项**：日常零散任务记录
- **今日计划**：当天具体执行计划
- **本周目标**：周度宏观目标设定
- 支持父子任务关联，形成目标树状结构
- 可视化进度追踪，清晰了解完成情况
- 智能截止时间提醒（紧急/警告/正常三级标识）

### ⏳ 番茄工作法专注计时器
- 可自定义专注时长（1-120分钟）
- 完整的开始/暂停/重置控制
- 专注历史统计面板：
  - 今日专注时长
  - 本月累计专注
  - 年度专注统计
  - 总体专注时间
- 专注完成后自动统计数据，并伴有视觉反馈

### 🌤️ 实时天气信息
- 基于高德地图API的精准天气数据
- 支持城市切换功能
- 智能出行建议提示
- 天气图标动态展示

### 🎧 其他实用模块
- **微博热搜**: 实时热门话题跟踪（多源备用机制）
- **热点资讯**: 科技、财经、体育等分类新闻
- **QQ音乐**: 快速跳转至音乐平台

## 技术实现

### 前端技术栈
- HTML5 + CSS3 + 原生 JavaScript
- 响应式布局设计，支持移动端访问
- 使用 Phosphor Icons 图标库
- CSS Grid 和 Flexbox 布局
- 玻璃拟态 (Glassmorphism) 设计风格

### 核心特性
- 所有数据本地存储（localStorage）
- 动态背景动画效果
- 丰富的交互反馈（悬停、点击、过渡动画）
- 模态对话框设计统一且美观
- 自适应不同屏幕尺寸

### 第三方服务集成
- 高德地图 Web 服务 API（天气信息）
- 微博热搜数据接口（多源备用机制）

## 使用说明

1. 直接打开 `index.html` 即可使用
2. 所有数据均保存在浏览器本地存储中
3. 首次使用时会加载默认快捷方式配置
4. 可通过界面操作完成所有设置和管理任务

### 快捷键操作
- 在任务输入框按下 Enter 键快速添加任务
- 在城市搜索框按下 Enter 键快速搜索城市

## 数据存储结构

### 快捷方式数据
```json
[
  { "title": "网站名称", "url": "网址" }
]
```

### 任务数据结构
```json
{
  "id": "唯一标识",
  "text": "任务内容",
  "type": "任务类型(todo/day/week)",
  "done": "是否完成",
  "deadline": "截止时间",
  "parentId": "父任务ID",
  "contribution": "对父任务贡献度",
  "legacyProgress": "历史保留进度"
}
```

### 专注统计数据
```json
{
  "日期": 秒数
}
```

- Firefox 65+

- Edge 79+

## 注意事项

1. 本应用完全基于前端技术构建，无需后端服务器支持
2. 所有数据存储于浏览器本地，清除浏览器数据会导致信息丢失
3. 如需更换高德地图 API Key，请修改代码中的 `AMAP_KEY` 常量
4. 微博热搜模块采用第三方免费接口，可能存在不稳定情况

## 开发与定制

您可以自由修改源代码以满足个性化需求：

- 更改主题颜色：调整 `:root` 中的 CSS 变量
- 修改默认快捷方式：编辑 `defaultShortcuts` 数组
- 调整番茄钟默认时长：修改初始化的 `defaultTime` 值
- 替换天气服务提供商：修改 `fetchWeather` 函数相关逻辑
